= EZ Common

=== EZ系列的公共模块。

== 功能

. Json与Scala对象转换（JsonHelper）
. Scala Bean操作（BeanHelper）
. Scala Class扫描操作（ClassScanHelper）
. Shell脚本操作（ShellHelper）
. 加密（EncryptHelper）
. 请求响应模型（ReqRespVO）

== 使用

[source]
----
<dependency>
    <groupId>com.ecfront</groupId>
    <artifactId>ez-common</artifactId>
    <version>1.0.0</version>
</dependency>
----
